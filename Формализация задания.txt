Базовый уровень
	Задача:
	Распределить количество поездов по поездоучасткам таким образом,
	чтобы их суммарная величина по сети была максимальной.
	Ограничения:
	- количество поездов входищих в станцию равно количеству поездов
	  выходящих из станции;
	- количество поездов проходящих по поездоучастку не превышает предельное
	  значение для поездоучастка (quantity_limit)

Средний уровень
	Задача:
	Распределить грузопотоки по поездоучасткам таким образом,
	чтобы средняя величина грузопотока приходящаяся на 1 километр пути
	по сети была максимальной.
	Ограничения:
	- из базового уровня;
	- тоннаж одного поезда не превышает предельное
	  значение (max_train_tonnage);
	- суммарный тоннаж по поездоучастку не превышает предельное
	  значение (tonnage_limit);
